視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓背景以比前景更慢的速度移動，營造 3D 深度感，帶給訪客更沉浸的瀏覽體驗。 視差效果源自光學錯覺：因為我們的眼睛會將較近的物體看得比遠處更大，也就會覺得遠處的物體移動得較慢。 這種錯覺早已在不同媒介中被運用，形成逼真的效果，最早可以追溯到迪士尼《白雪公主與七矮人》等傳統動畫， 以及《超級瑪利歐》等遊戲。隨著 CSS 與 HTML 的進步，視差效果也逐漸演變成今日網頁設計常見的技術。

其中一個早期的視差滾動網頁實作，是開發者 Glutnix 在 2007 年於部落格分享的範例與示範，使用 JavaScript 和 CSS2，就能在當時的 IE6 等瀏覽器上運作。不過直到 2011 年、HTML5 與 CSS3 普及後，這項技巧才真正受到網頁設 計師歡迎。支持者認為，這是一種擁抱網頁流動性的簡單方法，可運用視差背景更好地吸引使用者並提升整體體 驗。然而，普渡大學在 2013 年的研究指出：雖然視差滾動增強了部分使用者體驗，但並不一定能提升整體使用者 滿意度。


為什麼要使用視差圖像？

    設計師應該嘗試視差圖像的兩大原因：

  • 能幫助敘事

    視差圖像非常適合講故事。好的設計重在溝通——設計網站時，我們希望向訪客傳達產品、服務或品牌的 動人故事。視差滾動能讓訪客沉浸在獨特的線上體驗中，甚至可以打造單頁網站，讓讀者不用跳轉頁面就能 一路閱讀完整故事。

  • 有助提升互動

    視差圖像也能提升使用者互動。降低網站的跳出率是設計時的重要目標，因為高跳出率會影響搜尋排名。 精心打造的視差頁面能帶來動態體驗，讓使用者更願意停留。


想在網站加入視差滾動？可以參考以下建議：

  • 挑重點使用

    視差滾動用在特定元素時效果更好，不必所有頁面都套用。可以放在頁首或標題，或僅用在首頁，讓視差為 設計加分，而非分散注意力。

  • 壓縮圖片

    視差效果常會用到多種媒體與 CSS 形狀來營造深度。請壓縮圖片，避免檔案過大造成載入緩慢，影響效能。

  • 玩轉色彩

    視差滾動不只關乎前後景的圖片，色彩也很重要。背景可用較低彩度的色調，前景則運用鮮明色彩，營造更 強的立體感。
